# Votely 系統測試計畫書 (Test Plan)

本文件旨在提供 Votely 影音投票平台的驗收測試流程 (UAT)。請管理員或測試人員依照以下腳本逐步執行，以確保系統功能符合需求。

## 測試環境
- **網址**: (請填入您的測試站點或正式站點 URL)
- **瀏覽器**: 建議使用 Chrome 或 Edge (可使用無痕視窗模擬不同使用者)

## 測試角色 (User Roles)
1.  **影片上傳者 (Creator)**: 負責上傳參賽影片。
2.  **投票者 (Voter)**: 一般使用者，參與投票。
3.  **管理員 (Admin)**: 系統管理者，擁有最高權限。

---

## 測試腳本 (Test Scenarios)

### 1. 影片上傳者 (Creator) 流程
**目標**: 驗證上傳功能及權限限制 (無法自行修改/刪除)。

| 步驟 | 動作描述 | 預期結果 | 狀態 (Pass/Fail) |
|:---:|:---|:---|:---:|
| 1.1 | 開啟網站，在登入頁面輸入 **Creator Email** (例如: `creator@test.com`) 並登入。 | 成功登入，進入首頁 (Feed)。 | |
| 1.2 | 點擊右上角 **"+ Create"** 按鈕。 | 進入上傳頁面。 | |
| 1.3 | 選擇一個影片檔案，輸入 **標題 (Title)** 與 **敘述 (Description)**，點擊 **"Upload Video"**。 | 顯示上傳進度，成功後自動跳轉回首頁，並看到剛上傳的影片。 | |
| 1.4 | 點擊剛上傳的影片進入 **影片詳情頁 (Detail Page)**。 | 影片可正常播放，顯示正確的標題與敘述，作者名稱顯示為 Email。 | |
| 1.5 | **權限驗證**: 在詳情頁或首頁卡片上，尋找是否有「編輯 (Edit)」或「刪除 (Delete)」按鈕。 | **不應出現**任何編輯或刪除按鈕。 (若需修改需聯繫管理員) | |
| 1.6 | 登出 (點擊頭像 -> Logout)。 | 回到登入頁面。 | |

---

### 2. 投票者 (Voter) 流程
**目標**: 驗證投票功能及「一人一票」限制。

| 步驟 | 動作描述 | 預期結果 | 狀態 (Pass/Fail) |
|:---:|:---|:---|:---:|
| 2.1 | 開啟網站 (或使用無痕視窗)，輸入 **ASUS Email** (例如: `employee@asus.com`) 並登入。 | 成功登入，進入首頁。 | |
| 2.2 | 瀏覽影片列表，點擊進入 **影片 A**。 | 進入影片 A 詳情頁。 | |
| 2.3 | 點擊 **"Vote for this Video"** 按鈕。 | 按鈕變為紅色 **"Vote Cast"**，票數計入。 | |
| 2.4 | 返回首頁，點擊進入 **影片 B** (若無影片 B，請先用 Creator 帳號多上傳一部)。 | 進入影片 B 詳情頁。 | |
| 2.5 | 點擊 **"Vote for this Video"** 按鈕 (投票給影片 B)。 | 按鈕變為紅色 **"Vote Cast"**。 | |
| 2.6 | 返回 **影片 A** 查看狀態。 | 影片 A 的投票按鈕應恢復為未投票狀態 (白色)。驗證系統自動取消了上一票，確保**一人只能投一票**。 | |
| 2.7 | 登出。 | 回到登入頁面。 | |

---

### 3. 管理員 (Admin) 流程
**目標**: 驗證後台管理、數據查看及刪除功能。

| 步驟 | 動作描述 | 預期結果 | 狀態 (Pass/Fail) |
|:---:|:---|:---|:---:|
| 3.1 | 開啟網站，輸入管理員帳號 **`admin@votely.com`** 並登入。 | 成功登入，右上角出現 **"Admin"** 按鈕。 | |
| 3.2 | 點擊右上角 **"Admin"** 按鈕。 | 進入管理員儀表板 (Dashboard)。 | |
| 3.3 | **查看數據**: 檢查 "Total Videos" 和 "Total Votes" 數字。 | 數字應與目前系統內的影片數與總票數相符。 | |
| 3.4 | **查看排名與投票者**: 在下方列表中找到有被投票的影片，將滑鼠游標移到 **"Total Votes"** 的數字圓圈上 (Hover)。 | 應顯示浮動視窗 (Tooltip)，列出所有投票給該影片的 **投票者 Email** (例如看到 `employee@asus.com`)。 | |
| 3.5 | **隱藏影片**: 在列表中點擊某影片右側的 **眼睛圖示 (Hide)**。 | 圖示變更，影片被標記為 Hidden。回到首頁確認該影片已不顯示於公開列表。 | |
| 3.6 | **刪除影片**: 在列表中點擊某影片右側的 **垃圾桶圖示 (Delete)**，並確認刪除。 | 影片從列表中消失。數據統計自動更新。 | |
